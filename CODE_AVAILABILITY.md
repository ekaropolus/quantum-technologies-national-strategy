# Code Availability

This repository contains the source manuscripts, processed data, BibTeX bibliography, figure-generation code, and build script used to generate the accompanying PDF, TeX, and DOCX research outputs.

The repository supports:

- inspection of processed OpenAlex CSV extracts;
- regeneration of SVG and PNG figures;
- reconstruction of manuscript outputs from Pandoc Markdown and BibTeX sources;
- verification of citation generation through Pandoc citeproc.

The repository does not claim to provide a comprehensive global deployment dataset for quantum technologies. The quantitative data are descriptive proxies used for research synthesis and should be interpreted according to the limitations stated in the manuscripts.
